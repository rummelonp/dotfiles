# dotfiles

## Install

```sh
git clone git://github.com/mitukiii/dotfiles.git

# Link dotfiles
dotfiles/install.sh

# Install homebrew
dotfiles/homebrew/install.sh

# Install homebrew packages
eval "$(/opt/homebrew/bin/brew shellenv)"
brew bundle --global
```

## Uninstall

```sh
# Unlink dotfiles
dotfiles/uninstall.sh
```
